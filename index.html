CineSafe

Building a Content Warning and Accessibility Dataset for Movies

Data Engineering Natural Language Processing Machine Learning Accessibility Social Good

1. Project Overview

Movie platforms typically provide only broad ratings such as PG-13 or R. These ratings often fail to address the needs of:

  • Parents selecting movies for children
  • People with PTSD or trauma triggers
  • People sensitive to flashing lights
  • People with sensory sensitivities
  • Deaf and visually impaired viewers

For example, a viewer may be comfortable with an R-rated film but wish to avoid:

  • Sexual assault scenes
  • Self-harm content
  • Animal cruelty
  • Excessive jump scares
  • Strobe effects

Similarly, a visually impaired viewer may specifically need:

  • Audio descriptions
  • Closed captions

CineSafe aims to automatically generate detailed content warnings and accessibility metadata for movies.

2. Research Question

Can machine learning automatically generate meaningful content warnings and accessibility metadata from existing movie information?

3. Project Objectives

  1. Create an augmented movie dataset.
  2. Build machine learning models that predict content warnings.
  3. Develop accessibility metadata.
  4. Create a personalized recommendation engine.
  5. Provide an interactive dashboard for end users.

8. Dataset Augmentation Pipeline

  1. Collect movie metadata.
  2. Collect subtitles.
  3. Manually annotate 300–500 movies.
  4. Train machine learning models.
  5. Generate labels for the full dataset.
  6. Build recommendation and dashboard systems.

10. Machine Learning Problem

The project is formulated as a multi-label classification problem. 

Input:
Movie Metadata + Subtitles

Output:
Content Warnings + Accessibility Labels

Models

  • Logistic Regression
  • Random Forest
  • XGBoost
  • Fine-tuned BERT
